# coatless-py-pkg.github.io

Auto-built project index for [coatless-py-pkg](https://github.com/coatless-py-pkg), deployed to
[py-pkg.thecoatlessprofessor.com](https://py-pkg.thecoatlessprofessor.com/).

Source: [coatless/quarto-org-index](https://github.com/coatless/quarto-org-index).

Rebuilt daily by GitHub Actions plus on push to `main` and on
`repository_dispatch` event `refresh`.
